# Telegram Credit Bot

Educational Telegram bot for credit history tips.

## Setup
1. Get Telegram token from @BotFather.
2. Deploy to Render.com.
3. Set TELEGRAM_TOKEN in Render env vars.
4. Set webhook: https://api.telegram.org/bot<TOKEN>/setWebhook?url=<RENDER_URL>/<TOKEN>

## Files
- bot.py: Main script
- requirements.txt: Dependencies
